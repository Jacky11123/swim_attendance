# 🏊 游泳教學管理系統 v10

高中體育課游泳教學專用的出缺席與退費計算工具，單一 HTML 檔案，不需安裝、不需伺服器，直接用瀏覽器開啟即可使用。

---

## 功能總覽

| 步驟 | 功能 |
|------|------|
| 1 匯入名單 | 支援 Excel（.xlsx）、CSV、或直接貼上文字 |
| 2 出缺席登記 | 多班切換、逐一點名或批次補登；手機顯示選擇彈窗 |
| 3 費用計算 | 依出席次數自動計算退費，支援個別學生覆蓋金額 |
| 4 匯出清冊 | 列印 / 存 PDF / 匯出 CSV；支援上呈版（含簽名欄） |
| 設定 | 備份與還原、班級資料包交換（老師 → 體育組）、學期封存 |

---

## 使用方式

1. 下載 `swim_attendance.html`
2. 用瀏覽器開啟（建議 Chrome / Safari）
3. 依畫面步驟操作：設定課程 → 匯入名單 → 點名 → 計算費用 → 匯出

> **手機使用**：支援 Chrome Android、Chrome iOS、Safari iOS。  
> 讀取 Excel 需要網路（載入 SheetJS 元件）；若離線使用請改用「貼上文字」方式匯入名單。

---

## Excel 匯入規格

系統會自動辨識標題列，只需包含以下四欄（欄名可中英文、大小寫不限）：

| 欄位 | 可接受的欄名 |
|------|------------|
| 學號 | 學號、學生學號、studentid |
| 班級 | 班級、班別、class |
| 座號 | 座號、座次、號次、seatno |
| 姓名 | 姓名、學生姓名、name |

- 只讀取名稱為「**列印**」的工作表（Sheet）
- 底部合計列、簽名列等雜列會自動過濾

---

## 資料儲存說明

- 所有資料儲存在**本機瀏覽器的 localStorage**
- 換手機、換瀏覽器、或清除瀏覽器資料後資料會消失
- 請定期至「設定」頁面下載 JSON 備份檔

---

## 班級資料包交換（多老師協作）

適用情境：每位老師負責一至數班，最後由體育組彙整。

1. **老師端**：在「設定」→「繳交班級資料包」匯出各班 JSON
2. **體育組端**：在「設定」→「匯入並彙整」一次匯入多個 JSON

系統會自動驗證學年度、課堂日期、費用設定是否一致，不一致的班級會提示略過。
